# Google Play Store Apps Data Science Project

This project extracts, cleans, analyzes, and visualizes data from the provided Google Play Store Apps PDF.

## Steps
1. Extract data from PDF (Google Play Store Apps.pdf)
2. Clean and structure the data
3. Perform exploratory data analysis (EDA)
4. Visualize insights

## Requirements
- Python 3.7+
- pandas
- matplotlib
- seaborn
- pdfplumber

## Usage
1. Place the PDF file in the project directory.
2. Run the notebook or scripts to process and analyze the data.
